## üìÇ List of Practicals

| # | Practical Name | Description |
|:-:|:---|:---|
| **1** | **Hello World App** | A simple application displaying "Hello World" at the center of the screen. Tested on both Emulator and Physical Device. |
| **2** | **Activity Lifecycle** | Demonstrates the various Android lifecycle phases (`onCreate`, `onStart`, `onResume`, etc.) using logs or toast messages. |
| **3** | **Explicit Intent (Data Transfer)** | Two-activity app: Input text in Activity A and display it in Activity B upon clicking "Send". |
| **4** | **Implicit Intent (Share Action)** | Uses an implicit intent with the `SEND` action to allow the user to choose an app to share/display text input. |
| **5** | **Basic Calculator** | Performs arithmetic operations (Addition, Subtraction, Multiplication, Division) on user input. |
| **6** | **Spinner & Images** | Populates a Spinner from `strings.xml`. Displays a corresponding image from `res/drawable` based on the selection. |
| **7** | **Discount Calculator** | Calculates bill discounts (10%, 15%, 20%) using a `RadioGroup` and displays the final amount. |
| **8** | **Course Selection (RadioGroup)** | Displays a list of college courses. Selecting a course shows its Total Instructional Credit (TIC) in a TextView. |
| **9** | **Shopping List (Checkboxes)** | A checklist app where selected shopping items are dynamically displayed in a TextView. |
| **10** | **Login & Registration** | User authentication system with a Registration page. Shows a "Welcome User" pop-up upon successful login. |
| **11** | **Login with Logout Dialog** | Authenticates user and navigates to a Home Activity. The "Logout" button triggers an Alert Dialog (OK/Cancel) to confirm exit. |
| **12** | **Option Menu** | Implements an options menu with five items. Displays the name of the selected menu item in a TextView. |
| **13** | **LinearLayout & Key Events** | Converts text to uppercase on button click using `LinearLayout`. Also handles key events directly in `EditText` without button presses. |
| **14** | **TableLayout & Custom Styles** | Uses `TableLayout` to organize UI elements. Implements a custom `styles.xml` to apply specific styling to TextViews. |
| **15** | **SQLite CRUD Operations** | A complete database app allowing users to Create, Read, Update, and Delete records using SQLite. |
| **16** | **Background Color Changer** | Displays three vertically aligned buttons. Clicking a button changes the screen's background color accordingly. |

## üöÄ How to Run
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ShuklaCoder/Mobile-Application-Development-Material.git
    ```
2.  **Open in Android Studio:**
    * Launch Android Studio.
    * Select **Open an Existing Project**.
    * Navigate to the cloned folder and select a specific practical directory.
3.  **Build and Run:**
    * Let Gradle sync finish.
    * Connect your device or start the Emulator.
    * Click the **Run** (Play) button.
